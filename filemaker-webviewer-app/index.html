FileMaker WebViewer App

Welcome to Your FileMaker WebViewer App

This is a single-page application built with Vite, PostCSS, and Tailwind CSS.

Integration with FileMaker:

  • Use window.receiveFromFileMaker(data) to receive data from FileMaker
  • Call window.FileMaker.PerformScript('ScriptName', 'parameter') to run FileMaker scripts

Development Mode

Run npm run dev to start the development server.

Run npm run build to build for production.
